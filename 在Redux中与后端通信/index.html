# 准备

这里使用 json-server 模拟后端数据， json-server 是一个基于 JSON 的 HTTP 服务器，用于快速创建 RESTful API。

在 reducer 中，我们使用 redux-thunk 来处理异步操作， redux-thunk 是一个中间件，它允许我们返回一个函数而不是一个对象，并在函数中可以进行异步操作。
