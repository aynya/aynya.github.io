# 注册并获取 API Key

访问 https://openweathermap.org/api 注册账号。记得在邮箱验证，生成一个新的 API 密钥

# 通过 axios, async/await 获取天气信息

替换 YOUR_API_KEY 为 API 密钥
在 fetchWeather 函数中，我们使用 axios.get() 方法发送一个 HTTP GET 请求到 OpenWeatherMap API，并传递了 API 密钥和查询参数。
天气数据被存储在 weather 状态变量中。
